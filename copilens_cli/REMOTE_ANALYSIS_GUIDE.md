## Quick Commands

### 1. Quick Scan (No API Key Needed)

```powershell
copilens remote quick https://github.com/user/repo
```

Shows:
- Project type and architecture  
- Languages and frameworks
- File counts and code stats
- Quality score

### 2. Full AI Analysis (Gemini API Key Required)

```powershell
# Set API key first
$env:GEMINI_API_KEY="your-key"

# Comprehensive analysis
copilens remote analyze https://github.com/user/repo

# Security-focused
copilens remote analyze https://github.com/user/repo --type security

# Code quality analysis
copilens remote analyze https://github.com/user/repo --type quality

# Architecture analysis
copilens remote analyze https://github.com/user/repo --type architecture
```

## What Makes This Powerful?

### 1. Gemini 3 Pro with Deep Thinking
- **HIGH-level thinking** enabled

### 2. Google Search Integration
- Finds context about the repository

### 3. AI-Generated Code Detection
Copilens can estimate if code was AI-generated by analyzing:
- Comment verbosity and style
- Consistent formatting patterns
- Generic variable names

### Set in PowerShell

```powershell
# Current session
$env:GEMINI_API_KEY="your-key-here"

# Permanent
[System.Environment]::SetEnvironmentVariable('GEMINI_API_KEY', 'your-key', 'User')
```

### Or use .env file

Create `.env` in your project:
```
GEMINI_API_KEY=your-key-here
```

## Troubleshooting

### "Git not found"
```powershell
# Install Git
winget install Git.Git

# Or download from https://git-scm.com/downloads
```

### "Clone timeout"
Repository is too large. Copilens uses shallow clone with 5-minute timeout.

### "No LLM provider available"
```powershell
# Set API key
$env:GEMINI_API_KEY="your-key"

# Verify
python -c "import os; print(os.getenv('GEMINI_API_KEY'))"
```

### Rate Limits
Gemini Free Tier: 15 requests/minute, 1500 requests/day

Use `--type` for faster, smaller analyses to stay within limits.
