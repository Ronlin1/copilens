## Quick Start

### Web Application

```bash
cd copilens-web
npm install
# Add VITE_GEMINI_API_KEY to .env
npm run dev
```

Visit: http://localhost:5173

### CLI Tool

```bash
cd copilens_cli
pip install -r requirements.txt
pip install -e .
copilens --help
```
